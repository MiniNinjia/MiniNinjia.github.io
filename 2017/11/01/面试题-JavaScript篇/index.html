谈谈对this的理解。

this表示当前对象，this的指向是根据调用的上下文来决定的，默认指向window对象，指向window对象时可以省略不写。

全局变量

  1. 全局环境就是在里面，这里的this始终指向的是window对象，

局部环境

  1. 在全局作用域下直接调用函数，this指向window

  2. 对象函数调用，哪个对象调用就指向哪个对象

  3. 使用 new 实例化对象，在构造函数中的this指向实例化对象。

  4. 使用call或apply改变this的指向

加分项

  1. 用于区分全局变量和局部变量，需要使用this

  2. 返回函数当前的对象，jquery。

  1. 将当前的对象传递到下一个函数，看jquery1.8.3源码。
